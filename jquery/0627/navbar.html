Navigation's

필요 선택자

  • $('tag name')/$('.class name')/ $('#id')
  • 하위요소(자손)찾기 - selector.find('.sub') : 전체 풀다운 메뉴 제작시
  • 하위요소(자식)찾기 - selector.find() / selector.children('.sub') : 개별 풀다운 메뉴 제작시
  • 동위 요소 찾기(다음개체) - selector.next() : 클릭이벤트 개별메뉴 제작시
  • 동위요소찾기(개체선택반전) - selector.siblings() : 세로메뉴 개별 반응 제작시
  • 동위요소찾기(특정개체) - selector.filter('.on') : 개별메뉴 특정요소(클래스) 검색 선택
